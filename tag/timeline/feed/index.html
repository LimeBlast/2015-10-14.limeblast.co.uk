]]> Seeing as Facebook’s Timeline feature is making the move to Pages, I figured that now was a good time to share the Facebook Timeline Hack Template that I used to style up my Timeline Page.

The download includes a Photoshop template file which gives you a mock-up of Timeline header to style up, and an action file which allows you to save the separate header and avatar files ready for uploading to Facebook.

As you can see above, even someone with zero design and graphical talent, such as myself, is able to put something rather attractive together.
